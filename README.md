# Guide

## Used Tech

- Next.js 15 with trpc & react-query
- PostgreSQL
- Resend
- Uploadthing

## Customize Template

1. Add .env variables

- create db -> run pnpm push && pnpm generate
- create resend api key
- create uploadthing api key

2. update config.ts
3. add public/logo.png & public/auth-image.png & src/app/favicon.ico
